 => greater than or equal to
 <= less than or equal to
 == equality 
 != not equal
 === strict equality
 !== strict non-equality

These return a boolean value. 

e.g. 
```
age = 31;
age >= 18
true
```

You can compare two strings:
* Note this is based on Unicode

```
'a' < 'b'; //true

'A' > 'a'; //false
```
